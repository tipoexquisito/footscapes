> The framework does not prescribe what to teach. It describes what learning looks like when it is working.


21st Century Learning Design (21CLD) is a research-based framework developed by Microsoft Partners in Learning in collaboration with SRI International (2011). It provides educators and researchers with structured rubrics for evaluating how well a learning activity develops the competences students need in contemporary professional and civic life.

---

## The Six Dimensions

**Collaboration** — the extent to which students work together in ways that require genuine interdependence. The framework distinguishes real collaboration, where the product fails if any member does not contribute, from parallel group work, where individual outputs are merely added together. Maximum score: 5.

**Knowledge Construction** — the extent to which students build new understanding rather than reproduce existing information. Assessed by whether students apply knowledge to new contexts, generate interpretations, or produce original analysis. Maximum score: 5.

**Self-Regulation** — the extent to which students plan their own work, set goals, monitor progress and iterate. A high score requires that students have had prior opportunities to practice the skills they now apply independently. Maximum score: 4.

**Real-World Problem-Solving** — the extent to which students engage with authentic problems that have impact beyond the classroom. The highest scores require problems identified by students themselves, solutions with real-world implementation, and a genuine external audience. Maximum score: 4.

**Skilled Communication** — the extent to which students communicate for a specific purpose and audience, using evidence and multiple modes. Maximum score: 4.

**ICT for Learning** — the extent to which digital technologies are integral to the learning task, not supplementary to it. Maximum score: 5.

> [!concept] The 21CLD framework's most useful function is diagnostic: it reveals which design decisions produce which learning outcomes, and what would have to change to reach the next level. A score of 3 on Collaboration is not a failure — it is a description of where the ceiling was and why.
