# Game-Based Learning

Game-based learning uses the mechanics, structures and motivational dynamics of games — competition, progression, reward, challenge, narrative — as a framework for learning experiences.

It is distinct from [[Gamification]], which applies game elements to non-game contexts. Game-based learning uses actual game structures, whether digital or analogue, to drive the learning activity itself.

In this course, the quiz on Sunday evening — [[Building a European Dimension]] — functioned as game-based learning: points, teams, questions requiring active recall and discussion. The [[Images of Madrid]] walking tour used similar mechanics: eight locations, one point per photo, bonus points for creativity, Km0 as a collective goal. The game structure was light enough to motivate without overriding curiosity.
