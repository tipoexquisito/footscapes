Using the acronym PREP to organize your arguments:

- Point (opinion): 
	- In my opinion
	- I believe/ I reckon
	- I think
	- If you ask me,

- Reason: 
	- because...

- Example:
	- for example
	- for instance
	- such as
	- as an illustration
	- to demonstrate

- Point (conclusion):
	- in conclusion / to conclude
	- in short / to sum up
	- overall / to summarize
