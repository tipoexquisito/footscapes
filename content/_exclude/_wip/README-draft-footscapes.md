# Footscapes

A digital garden. Notes on learning experience design, active methodologies and vocational training, grown from an Erasmus+ KA1 mobility in Madrid and Toledo (May 2026) and from over a decade of practice as a vocational trainer and communication designer.

Live at [tipoexquisito.github.io/footscapes](https://tipoexquisito.github.io/footscapes).

## What this is

Not a blog, not a portfolio site in the traditional sense — a working collection of frameworks, reflections, field notes and case studies, some fully formed, some still seeds. Start at [About me](Home/About%20me.md) or the [index](index.md) for reading paths by interest.

## Structure

- `Home/` — entry points: About me, Portfolio, notes on process
- `Learning/` — notes on being taught: active methodologies, e-/b-learning design, frameworks, field learning projects
- `Experience/` — notes on learning in the world: case studies from vocational training practice (Transforma-te Transforma, Arco Maior, CASEL, ROAD)
- `Design/` — notes on designing what stays: visual and communication design work

## Recently added

- **Portfolio** (`Home/Portfolio.md`) — a working portfolio of trainer practice: experience, frameworks and a running selection of projects, most recently supporting a candidatura to Escola Profissional Raul Dória.

## Publishing changes

```
cd ~/Documents/footscapes
git add -A
git commit -m "descreve aqui o que mudou"
git push
```

GitHub Actions rebuilds and publishes the site within 2–3 minutes of a successful push. Check the **Actions** tab for a green check.
